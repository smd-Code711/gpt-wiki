# GPT Wiki

LLMが、追加された資料を読み取り、相互にリンクされたMarkdown形式のWikiとして継続的に整理・更新するためのリポジトリです。

Andrej Karpathyの [LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) の考え方を参考に、最小構成で開始します。

## 構成

- `raw/`: 人が追加する原資料。LLMは内容を変更しません。
- `wiki/`: LLMが生成・更新するWikiページ。
- `wiki/index.md`: Wiki全体の索引。
- `wiki/log.md`: 取り込み、質問対応、点検の履歴。
- `AGENTS.md`: LLMが従う運用ルール。

## 基本的な使い方

1. 原資料を `raw/` に追加します。
2. LLMに資料の取り込みを依頼します。
3. LLMが `wiki/` のページ、索引、関連リンク、履歴を更新します。
4. Wikiに対して質問したり、矛盾・孤立ページ・古い記述の点検を依頼します。

このリポジトリでは、LLMによる変更は実行前に利用者の承認を得る運用を前提とします。
