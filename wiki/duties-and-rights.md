# 服務と職員の権利・義務

地方公務員法第30条は、すべての職員が全体の奉仕者として公共の利益のために勤務し、職務遂行に当たって全力を挙げて専念すべきことを服務の根本基準とする。

## 主な服務義務

| 内容 | 主な条文 |
|---|---|
| 服務の宣誓 | 第31条 |
| 法令等および上司の職務上の命令に従う義務 | 第32条 |
| 信用失墜行為の禁止 | 第33条 |
| 秘密を守る義務 | 第34条 |
| 職務に専念する義務 | 第35条 |
| 政治的行為の制限 | 第36条 |
| 争議行為等の禁止 | 第37条 |
| 営利企業への従事等の制限 | 第38条 |

これらは一律ではない。教育公務員の政治的行為には教育公務員特例法の特例があり、企業職員や特定地方独立行政法人職員には地方公営企業法・地方独立行政法人法による適用除外や読替えがある。

## 権利・利益保護

給与、勤務時間その他の勤務条件は条例で定めることを基本とし（第24条・第24条の5）、勤務条件に関する措置要求や不利益処分への審査請求などの制度が設けられている（第46条以下）。労働基本権との関係は[[labor-rights-and-labor-laws]]を参照。
